# Number Guess Game

## Description
This is a Python game where the computer generates a random number and the user tries to guess it.

## Features
- Random number generation
- User input
- Guess checking
- Win/lose message

## Concepts Used
- random module
- while loops
- if-else conditions
- User input

## How to Run
1. Open number-guess-game.py
2. Run the program
3. Enter guesses until you find the correct number
